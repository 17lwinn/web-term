Johnny's Terminal
=================

A NodeJS based terminal that runs in browser as a local server. It uses [tty.js](https://github.com/chjj/tty.js).

## Installation

```sh
$ sudo npm install johnnys-terminal -g
```

Then run:

```sh
$ johnnys-terminal
```

and open `http://localhost:8080/` in your browser.
